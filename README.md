## What it does

Conversion flow engine with dynamic pricing and deal routing for real estate leads.

## Quick Start

`ash
git clone https://github.com/MrHTC/propertyvala-conversion-agent.git
cd propertyvala-conversion-agent
pip install -e .
python run.py
`

## Why we built it

Real estate agencies struggle with unqualified leads, inconsistent follow-up, and no systematic way to score buyer intent. This agent solves that by automating lead sourcing, qualification, and outreach with measurable metrics.

## Gaps it fills

- Replaces manual lead research and cold calling
- Provides consistent 0-100 scoring instead of gut-feeling
- Automates WhatsApp outreach without manual DMs
- Tracks conversions and commissions transparently

## System integration

Part of the [AGENTIC-LEAD-ENGINE](https://github.com/MrHTC/AGENTIC-LEAD-ENGINE) ecosystem. Works with propertyvala-leads-agent, propertyvala-qualification-agent, propertyvala-outreach-agent, propertyvala-sales-agent, propertyvala-analytics-agent, propertyvala-conversion-agent, and propertyvala-commission-agent.
